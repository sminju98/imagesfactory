## 🎯 타겟 고객 세분화

### 1차 타겟 (Primary)

#### 마케터 (30-40%)
- **연령**: 25-35세
- **직업**: 디지털 마케터, SNS 매니저
- **Pain Point**: 매일 많은 콘텐츠 이미지 필요
- **메시지**: "하루 30분이면 한 달치 이미지 완성"
- **채널**: 마케팅 커뮤니티, 블로그, 링크드인

#### 콘텐츠 크리에이터 (25-35%)
- **연령**: 20-30세
- **직업**: 유튜버, 블로거, 인스타그래머
- **Pain Point**: 썸네일, 포스팅 이미지 제작 시간
- **메시지**: "조회수 올리는 썸네일, AI가 만들어드려요"
- **채널**: 유튜브, 인스타그램, 크리에이터 커뮤니티

#### 디자이너 (20-30%)
- **연령**: 25-40세
- **직업**: 그래픽 디자이너, UI/UX 디자이너
- **Pain Point**: 레퍼런스, 프로토타입 제작
- **메시지**: "영감을 이미지로, 빠르게"
- **채널**: 노트폴리오, 비핸스, 디자인 커뮤니티

### 2차 타겟 (Secondary)

#### 쇼핑몰 운영자
- **연령**: 30-50세
- **직업**: 온라인 쇼핑몰 대표/담당자
- **Pain Point**: 상품 상세페이지 이미지 비용
- **메시지**: "월 100만원 이미지 제작비 → 10만원으로"

#### 스타트업 팀
- **연령**: 25-40세
- **직업**: 스타트업 팀원
- **Pain Point**: 제한된 예산, 빠른 실행
- **메시지**: "디자이너 없어도 괜찮아요"

## 🚀 출시 전략

### Phase 1: 클로즈드 베타 (2주)

**목표**: 50명의 얼리어답터 확보, 피드백 수집

**전략**:
1. **리크루팅**
   - 트위터, 링크드인에서 초대
   - 디자이너/마케터 커뮤니티 포스팅
   - 지인 네트워크 활용

2. **인센티브**
   - 3,000 포인트 무료 제공 (일반 3배)
   - 평생 20% 할인 쿠폰
   - 베타 테스터 명예의 전당

3. **피드백 수집**
   - 매일 사용 후기 요청
   - 1:1 인터뷰 진행
   - 개선 사항 즉시 반영

### Phase 2: 오픈 베타 (1개월)

**목표**: 500명 확보, 입소문 확산

**전략**:
1. **프로덕트 헌트 코리아 론칭**
   - 한국어 소개 페이지
   - 데모 영상 제작
   - 론칭 이벤트 (추가 포인트)

2. **커뮤니티 마케팅**
   - 디스쿼이엇: 제품 소개 및 피드백
   - 커리어리: 마케터 타겟 포스팅
   - 오픈카톡방: 사용자 커뮤니티 운영

3. **레퍼럴 프로그램**
   - 친구 초대 시 양쪽 1,000 포인트
   - 초대 순위 공개
   - 상위 10명 추가 보상

### Phase 3: 정식 출시 (지속)

**목표**: 월 1,000명 이상 신규 가입

**전략**:
1. **프레스 릴리스**
   - IT 매체 (테크크런치 코리아, 벤처스퀘어)
   - 경제 매체 (한국경제, 조선비즈)
   - 보도자료 배포

2. **유료 광고 시작**
   - 구글 검색 광고
   - 페이스북/인스타그램 광고
   - 유튜브 광고

3. **콘텐츠 마케팅 강화**
   - 블로그 운영
   - 케이스 스터디 발행
   - 사용 가이드 영상
