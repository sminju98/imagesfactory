## 📋 문서 개요

### 목적
본 문서는 imagesfactory 프로젝트의 비즈니스 요구사항을 정의하고, 프로젝트의 목표, 범위, 이해관계자, 비즈니스 가치를 명확히 하기 위해 작성되었습니다.

## 🎯 비즈니스 목표

### 주요 목표
1. **시장 진입**: AI 이미지 생성 시장에서 '대량 생성'과 '편의성'에 특화된 포지셔닝
2. **사용자 확보**: 출시 6개월 내 월간 활성 사용자(MAU) 10,000명 달성
3. **매출 목표**: 출시 1년 내 월 매출 2,000만원 달성
4. **시장 확대**: 한국 시장 안착 후 글로벌 시장 진출

### 비즈니스 가치
- **차별화 포인트**: 개별 이미지 생성이 아닌 배치(대량) 생성에 특화
- **타임 세이빙**: 디자이너, 마케터의 작업 시간 80% 단축
- **비용 효율**: 타 AI 이미지 생성 서비스 대비 30% 저렴한 가격
- **자동화**: 생성부터 이메일 전송까지 원스톱 프로세스
- **콘텐츠 팩토리**: 이미지뿐만 아니라 SNS 콘텐츠(릴스, 카드뉴스) 자동 제작
- **AI 통합**: 최신 AI 모델(Perplexity, GPT, Grok, Veo3) 활용한 지능형 콘텐츠 생성

## 🌍 시장 분석

### 타겟 시장
- **1차 시장**: 대한민국
  - 인구: 5,000만명
  - 디지털 콘텐츠 시장 규모: 연 10조원
  - AI 서비스 수용도: 높음

- **2차 시장** (6개월~1년 후): 글로벌
  - 영어권: 미국, 영국, 캐나다, 호주
  - 아시아: 일본, 대만, 싱가포르
  - 유럽: 독일, 프랑스

### 경쟁사 분석

#### 직접 경쟁사
| 서비스명 | 강점 | 약점 | 차별화 전략 |
|---------|------|------|------------|
| DALL-E (OpenAI) | 품질 우수, 브랜드 인지도 | 비싼 가격, 개별 생성만 가능 | 대량 생성 + 저렴한 가격 |
| Midjourney | 예술적 퀄리티, 커뮤니티 | 디스코드 필수, UI 복잡 | 웹 기반 간편 UI |
| Stable Diffusion | 오픈소스, 무료 | 기술적 진입장벽 높음 | 비개발자도 쉽게 사용 |
| Canva AI | 통합 디자인 툴 | AI 기능 제한적 | 이미지 생성에만 집중 |

#### 간접 경쟁사
- Getty Images AI
- Adobe Firefly
- Shutterstock AI

### 시장 기회
- AI 이미지 생성 시장 연평균 성장률: 35% (2023-2028)
- B2B 시장 (광고대행사, 기업 마케팅팀) 수요 증가
- 개인 크리에이터 경제 성장
- SNS 콘텐츠 수요 폭발적 증가
- **AI 영상 생성 시장**: 릴스/숏폼 콘텐츠 수요 급증 (2025년 기준)
- **자동화 콘텐츠 제작**: 마케터의 콘텐츠 제작 시간 단축 니즈

## 👥 타겟 고객

### 주요 페르소나

#### 1. 마케터 민지 (28세)
- **직업**: 스타트업 마케팅 매니저
- **Pain Point**: 
  - 매일 SNS용 이미지 10-20장 필요
  - 외주 비용 부담 (장당 5만원)
  - 빠른 턴어라운드 필요
- **Needs**: 
  - 저렴한 가격
  - 빠른 생성 속도
  - 일관된 스타일
- **예상 사용**: 주 3회, 회당 20장 생성
- **월 예상 매출**: 30,000원

#### 2. 디자이너 준호 (32세)
- **직업**: 프리랜서 그래픽 디자이너
- **Pain Point**: 
  - 프로토타입 제작에 시간 소요
  - 클라이언트 컨셉 설명용 이미지 필요
  - 레퍼런스 이미지 부족
- **Needs**: 
  - 다양한 스타일 옵션
  - 고해상도 이미지
  - 빠른 반복 작업
- **예상 사용**: 주 5회, 회당 15장 생성
- **월 예상 매출**: 50,000원

#### 3. 쇼핑몰 운영자 수진 (35세)
- **직업**: 온라인 쇼핑몰 대표
- **Pain Point**: 
  - 상품 상세페이지 이미지 제작 비용
  - 매일 신상품 이미지 필요
  - 제작 시간 부족
- **Needs**: 
  - 상품 배경 이미지
  - 썸네일 제작
  - 대량 생성
- **예상 사용**: 주 4회, 회당 30장 생성
- **월 예상 매출**: 80,000원

#### 4. 콘텐츠 크리에이터 지훈 (25세)
- **직업**: 유튜버 / 블로거
- **Pain Point**: 
  - 영상 썸네일 제작
  - 블로그 포스팅 이미지
  - 독창적인 이미지 필요
- **Needs**: 
  - 눈에 띄는 이미지
  - 빠른 제작
  - 저렴한 가격
- **예상 사용**: 주 2회, 회당 10장 생성
- **월 예상 매출**: 15,000원

### 세컨더리 고객
- 광고 대행사
- 기업 인하우스 마케팅팀
- 교육 콘텐츠 제작자
- 출판사
- 게임 개발사 (컨셉 아트)

#### 변동 비용
- AI API 비용: 
  - Stable Diffusion API: 이미지당 10-50원
  - 마진율: 80-90%

### 손익분기점 분석

**월 손익분기점 계산**
```
고정비: 16,300,000원
변동비: 매출의 15% (AI API + 인프라 + 수수료)

손익분기 매출 = 고정비 / (1 - 변동비율)
              = 16,300,000 / (1 - 0.15)
              = 19,176,470원

필요 포인트 판매: 약 40,000,000 포인트
필요 이미지 생성: 약 400,000장
필요 활성 유저: 약 2,000명 (1인당 월 200장 생성 가정)
```

## 🎯 성공 지표 (KPI)

### 비즈니스 KPI

#### 단계별 목표
| 시기 | 목표 |
|-----|------|
| 베타 출시 (1개월) | 500명 가입, 10,000장 생성 |
| 정식 출시 (3개월) | 2,000명 MAU, 월 매출 1,500만원 |
| 성장기 (6개월) | 5,000명 MAU, 월 매출 3,000만원 |
| 안정기 (12개월) | 10,000명 MAU, 월 매출 5,000만원 |

#### 주요 지표
1. **사용자 지표**
   - 신규 가입자 수 (주간/월간)
   - MAU (Monthly Active Users)
   - DAU/MAU 비율 (Stickiness)
   - 리텐션율 (D1, D7, D30)
   - 이탈율 (Churn Rate)

2. **수익 지표**
   - MRR (Monthly Recurring Revenue)
   - ARPU (Average Revenue Per User)
   - LTV (Lifetime Value)
   - CAC (Customer Acquisition Cost)
   - LTV/CAC 비율 (목표: 3 이상)

3. **사용 지표**
   - 일일 이미지 생성 수
   - 사용자당 평균 생성 수
   - 프롬프트 성공률
   - 재생성 비율
   - 다운로드율

4. **전환 지표**
   - 무료 → 유료 전환율 (목표: 15%)
   - 구매 전환율 (목표: 20%)
   - 재구매율 (목표: 60%)

5. **만족도 지표**
   - NPS (Net Promoter Score) (목표: 50+)
   - 고객 만족도 (CSAT) (목표: 4.5/5)
   - 추천율

- Week 3: 포인트 & 이메일

  - 대시보드

  - 반응형 디자인

## 🚀 출시 전략

### 한국 시장 진입 전략

#### 1단계: 얼리어답터 확보 (0-1개월)
- **타겟**: 테크 얼리어답터, 디자이너 커뮤니티
- **채널**: 
  - 프로덕트 헌트 코리아
  - 디자이너 커뮤니티 (브런치, 노트폴리오)
  - 트위터, 링크드인
- **전략**: 무료 포인트 넉넉하게 제공 (2,000 포인트)

#### 2단계: 입소문 확산 (1-3개월)
- **타겟**: 블로거, 유튜버, 인플루언서
- **채널**: 
  - 레퍼럴 프로그램
  - 인플루언서 협업
  - 케이스 스터디 공유
- **전략**: 추천인 보상 프로그램 (양쪽 모두 1,000 포인트)

#### 3단계: 대중화 (3-6개월)
- **타겟**: 일반 소비자, 중소기업
- **채널**: 
  - 페이스북/인스타그램 광고
  - 구글 검색 광고
  - 유튜브 광고
  - 네이버 블로그
- **전략**: 성공 사례 중심 마케팅

### 글로벌 확장 전략

#### 시장 우선순위
1. **1차 (6-9개월)**: 영어권

   - 캐나다
   - 영국
   - 호주

2. **2차 (9-12개월)**: 아시아
   - 일본 (높은 ARPU)
   - 대만
   - 싱가포르

3. **3차 (12-18개월)**: 유럽
   - 독일
   - 프랑스
   - 네덜란드

#### 현지화 요소
- 언어 번역 (UI/UX, 마케팅)
- 결제 수단 (Stripe, PayPal)
- 고객 지원 (현지 시간대)
- 문화적 고려 (프롬프트 예시, 이미지 스타일)
- 법적 컴플라이언스 (GDPR, CCPA)

## ⚖️ 법적/규제 고려사항

### 대한민국
- **개인정보보호법**: 사용자 동의, 데이터 처리 방침
- **전자상거래법**: 환불 정책, 약관

- **GDPR** (유럽): 데이터 보호, 쿠키 동의
- **CCPA** (캘리포니아): 개인정보 판매 금지
- **저작권**: 각국 AI 생성물 저작권 법 준수

### 서비스 이용약관 주요 사항
- AI 생성 이미지 사용권 명시
- 부적절한 프롬프트 금지
- 포인트 환불 정책
- 책임 제한 조항
- 생성 실패 시 포인트 환불
