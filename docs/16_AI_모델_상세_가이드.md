## 1. DALL-E 3 (OpenAI) 💰200pt

### 제작사
- **회사**: OpenAI
- **출시**: 2023년 9월
- **관계**: ChatGPT의 이미지 생성 모델

### 특징
- **ChatGPT 통합**: ChatGPT Plus/Pro에서 사용 가능
- **용도**: ChatGPT에서 "이미지 그려줘" 하면 DALL-E 3 사용
- **품질**: 현존 최고 수준의 품질과 프롬프트 이해도
- **안전성**: 강력한 콘텐츠 필터링

### 주요 사용처
- OpenAI ChatGPT Plus/Pro
- Microsoft Bing Image Creator
- Microsoft Designer
- 기업용 AI 솔루션

### 기술 스펙
- 해상도: 1024x1024, 1792x1024, 1024x1792
- API: OpenAI API
- 가격: $0.040/image (standard), $0.080/image (HD)

```

✅ 장점:
```

- **DALL-E**: Salvador Dalí (화가) + WALL-E (픽사 영화)

- **공식 API 없음**: Discord 봇만 제공

- **대안**: Flux, Leonardo가 유사한 예술성

### Stable Diffusion의 영향력
- **오픈소스 혁명**: AI 이미지 생성 대중화
- **파생 모델**: 수만 개의 커스텀 버전
- **Leonardo, Realistic Vision** 모두 SD 기반

- OpenAI 공동 창업자였다가 떠남
- ChatGPT/DALL-E의 경쟁자로 Grok/Aurora 개발
